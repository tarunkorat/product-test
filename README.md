## Requirement
1. PHP 8.2
2. Node 18+

## Setup

1. composer install
2. cp .env.example .env
3. php artisan key:generate
4. php artisan serve

## Data Storage
Data stored in: storage/app/products.json

## Features
- AJAX form submit
- JSON storage
- Totals & grand total
- Edit functionality

## Assumptions
- No authentication required
- Single-user environment
